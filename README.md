## Installation
Clone this repository and import into **Android Studio**

With SSH
```bash
git clone git@github.com:SummerList/app-android.git
```
With HTTPS
```bash
git clone https://github.com/SummerList/app-android.git
```

## Then

You have to do:
```bash
composer install
```

and:
```bash
symfony server:start
```
